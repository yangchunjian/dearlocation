                # 1. 传统方式1

                （执行一遍，一条条试）

                可尝试 python3 -m pip install --upgrade pip

                pip install cv2

                pip3 install opencv-python

                国内镜像

                pip3 install opencv-python -i http://pypi.mirrors.ustc.edu.cn/simple/open in new window

                pip install cv2 -i http://pypi.mirrors.ustc.edu.cn/simple/open in new window

                # 2. 第一阶段 终极办法01

                切换 python 版本， python3.9 不支持 cv2 (python3.7及以下)

                # 3. 第二阶段 终极办法02

                自己安装

                https://www.lfd.uci.edu/~gohlke/pythonlibs/#wordcloudopen in new window

                等待下载好，copy到python安装目录下的lib目录当中

                接着，在cmd中安装（先进入lib目录中再安装你下载的那个文件）

                pip install opencv_python‑2.4.13.7‑cp27‑cp27m‑win_amd64.whl

                等待安装好，验证一下。

                pip install numpy
